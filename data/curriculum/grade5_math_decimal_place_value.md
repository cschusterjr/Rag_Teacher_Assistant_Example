# Grade 5 Mathematics: Decimal Place Value

Subject: Mathematics
Grade: 5
Unit: Decimals
Lesson: Decimal Place Value
Estimated Time: 50 minutes

## Prerequisite Skills

Students should understand whole number place value through the millions place and be familiar with basic decimal notation using tenths.

## Learning Objectives

Students will identify the value of digits in decimal numbers through the thousandths place. Students will compare and write decimal numbers using place value understanding.

## Standards

Students extend the place value system to include decimals and explain the value of each digit based on its position.

## Materials

- Place value chart
- Base-ten blocks
- Decimal place value chart
- Whiteboard
- Student practice worksheet

## Lesson Overview

Students explore how the place value system extends beyond whole numbers into decimals. They use visual models and place value charts to compare and describe decimal numbers.

## Teacher Instructions

Review whole number place value before introducing tenths, hundredths, and thousandths. Demonstrate how moving one place to the right decreases a digit's value by a factor of ten.

Model several decimal numbers and ask students to explain the value of each digit.

## Guided Practice

Students complete place value charts for several decimal numbers. As a class, compare pairs of decimal numbers and explain which is greater using place value reasoning.

## Independent Practice

Students write decimal numbers in expanded form, compare decimal values, and identify the value of underlined digits.

## Differentiation

Provide place value charts and color-coded models for students needing additional support. Challenge advanced students to order sets of decimal numbers from least to greatest.

## Assessment

Students complete an exit ticket identifying the value of digits in decimal numbers and comparing two decimals using greater than, less than, or equal to.

## Vocabulary

- Decimal
- Tenths
- Hundredths
- Thousandths
- Place Value
- Expanded Form

## Extensions

Students create real-world examples involving decimal numbers, such as money or measurements, and explain the place value of each digit.
